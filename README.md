# Udemy Clone

A simple Udemy clone built using **HTML** and **CSS (Flexbox)**.  
This project focuses on clean layout design, responsive sections, and basic front-end structure without using any frameworks.

## Features
- Homepage layout inspired by Udemy
- Course section design
- Fully responsive using Flexbox
- Clean and beginner-friendly code

## Technologies Used
- HTML
- CSS (Flexbox)

## Purpose
Created to practice front-end development skills and layout building using pure HTML and CSS.
